# Image Gallery

A simple image gallery created using HTML, CSS, and JavaScript.  
It allows users to view images with navigation and smooth transitions.

## Features
- Grid-style image layout  
- Next/Previous navigation  
- Lightbox view for enlarged images  
- Smooth hover and transition effects  
- Responsive design

## Technologies Used
- HTML  
- CSS  
- JavaScript  

## How to Run
1. Download or clone the repository  
2. Open `index.html` in your browser
